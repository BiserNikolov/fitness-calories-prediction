# Fitness Calories Prediction :

Goal: Predict calories burned during workout sessions using physiological

and behavioral features from gym members.
## Data Sources

1. Gym Members Exercise Tracking Dataset - gym\_members\_exercise\_tracking.csv ->
 https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset

2. Exercise Dataset - exercise\_dataset.csv ->
https://www.kaggle.com/datasets/aakashjoshi123/exercise-and-fitness-metrics-dataset
## Tech Stack

Python · pandas · scikit-learn · matplotlib · seaborn · Jupyter
